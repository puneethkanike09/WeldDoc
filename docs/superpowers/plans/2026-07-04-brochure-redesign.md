# Brochure Redesign — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Redesign the WeldDoc 2-page A4 PDF brochure to beat the WeldEye competitor brochure using a full-bleed welding photo hero, Forge Steel color palette, and tighter layout.

**Architecture:** All brochure content lives in `src/lib/brochure/content.ts`. The React component `src/components/brochure/brochure-document.tsx` renders it. All visual styling is in `src/app/brochure/brochure.css`. PDF is generated by `scripts/generate-brochure-pdf.mjs` via Puppeteer.

**Tech Stack:** Next.js 14, React, Tailwind CSS (imported in brochure.css), Puppeteer (PDF), Unsplash CDN (stock photo)
